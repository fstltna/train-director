Select the interface you wish to use:

 

Single-page, tabbed interface

This user interface only shows one view at a time. You can select which view from the tabs at the top of the screen.
This user interface is ideal for small screens or for tablets/mobile devices.

 

 

 

Multi-pane windowed interface

This user interface lets you open multiple views to see the state of your simulation in one single browser window.
You can open, close and position the different views on the screen.
This user interface is ideal for desktop computers with large screens.

To activate the ability to view the layout from the web browser, you need to enable the appropriate option in the Preferences dialog.
Note however that enabling this option may slow down the computer significantly.
Therefore it is recommended that you enable this option only for small simulations, or if you have a very powerful computer.
