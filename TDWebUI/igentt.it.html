Questa pagina è usata per cercare di generare automaticamente i file orario per Train Director.

Le pagine web per le stazioni nel tracciato devono essere salvate nella cartella "$TDHOME/gentt/stations", dove $TDHOME è la cartella dove si trova l'eseguibile di traindir.

La pagina trova automaticamente le pagine salvate e cerca di acquisire l'orario per ogni treno presente in tali pagine.

Le pagine riconosciute sono:

  • Le pagine dal sito delle Ferrovie Federali Svizzere relative a partenze e arrivi
  • Le pagine delle stazioni listate sul sito e656.net.
